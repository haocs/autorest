# MS-Rest-Azure-Auth-Node

Authentication libarary for nodeJS Azure client libraries.

- **Node.js version: 4.x.x or higher**


## How to Install

```bash
npm install ms-rest-azure-auth-node
```

## Usage
```javascript
var msrestAzureAuth = require('ms-rest-azure-auth-node');
```
